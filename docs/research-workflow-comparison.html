Research brief · 22 July 2026

Can a Jump Model beat HMM and Buy & Hold?

We rebuilt Shu et al.'s market-or-cash idea on a past-only public proxy sample, then tested five deliberately simple Jump Model changes and one loss-scale diagnostic. The target is simple: higher net Sharpe than both benchmarks on the same dates.

Latest result: scaling the DD loss by three does not create a cross-market winner. Scaled DD still beats fixed JM Sharpe in all three markets, but beats both HMM and buy-and-hold only in the US. The result is not_supported; raw loss scale is not the main explanation for the DD result.

1. What is the paper's idea?

Market behavior appears to persist in favorable and unfavorable periods. HMM represents persistence with transition probabilities. The Jump Model instead clusters observations and charges a penalty when the state changes. Shu et al. ask whether that simpler persistent state signal improves a 0/1 investment rule: hold the market in the favorable state and local cash in the unfavorable state.

The paper is therefore not “fit JM and compare clusters.” The economic test is whether the resulting strategy beats both buy-and-hold and an HMM strategy after delay and costs.

2. The whole experiment

  1. 1Daily data

    Equity log return for HMM; downside-risk features for JM.

  2. 2Online models

    Fit HMM from returns and JM from features, using past data only.

  3. 3State

    Label the current observation favorable or unfavorable.

  4. 4Position

    Decide after t, trade at the close of t+1, and first earn at t+2.

  5. 5Test

    Charge 10 bps per change and compare net Sharpe with HMM and buy-and-hold.

How lambda is chosen: at each month-end, every candidate is replayed online over the preceding eight calendar years. Its validation strategy uses the same one-day trading delay and 10-bps cost; the candidate with the highest validation Sharpe is used for the next month. HMM smoothing is selected the same way.

This is the core procedure in Shu et al. Section 3.4.3. The exact candidate grids, 252-return eligibility floor, 10−12 tie rule, and 5% boundary diagnostic are repository conventions because the paper does not publish them.

4. Five simple changes we tested

All challengers keep the same causal timeline and training window of 3,000 complete observations, training window, January/July refits, one-day trading delay, 10-bps cost, and raw λ grid unless the card says otherwise. Only one idea changes at a time.

Fixed JM: minimize Σt Lt(st) + λ Σt≥1 1[st ≠ st-1]

1. Static λ = 50

Remove monthly λ selection and use the source-visible illustrative value 50 everywhere.

2. DD-only

Cluster only 10-day exponentially weighted downside deviation: squared negative excess returns, then a square root. Ask whether the two Sortino features add signal or mostly noise.

3. Confirm twice

Accept a new raw state only after seeing it on two consecutive observations. This is a simple anti-whipsaw rule.

4. Return-aware

During fitting, also group only matured t+2 excess returns (γ = 1). Online decoding still uses today's features only.

5. Robust L1

Replace squared distance and mean centers with absolute distance and median centers, reducing sensitivity to outliers.

The return-aware idea is no longer untested. It was implemented with γ = 1, checked against the exact γ = 0 fixed-JM route, and run in all three markets. It passed 0/3.

6. What does this mean?

  • The checked implementation path is internally consistent: losses, penalties, states, signals, positions, trades, costs, and returns were checked end to end.
  • Simplicity helped in one market: DD-only raised fixed-JM Sharpe by +0.337682 / +0.060001 / +0.094622 in US / DE / JP, but crossed both benchmarks only in the US.
  • The market objective is not met: no tested JM beats both benchmarks in all three markets. This is development evidence, not a profitability claim.

The loss-scale control retained the DD clue: multiplying DD loss by three moved Sharpe by only −0.023 / −0.031 / +0.004 in US / DE / JP, and scaled DD still beat fixed-JM Sharpe in all three. That makes feature removal the stronger explanation than the original raw loss scale, but does not prove that Sortino features are harmful. The selector moved raw λ upward in 73.7% / 81.3% / 66.1% of paired months and selected the upper endpoint in 0% / 22.9% / 29.0%; the finite grid still binds in DE and JP.

One-state-collapse caveat: some fitted two-state candidates left only one reachable state. Collapsed fit×λ rows (US / DE / JP) were: DD-only 95 / 132 / 124; return-aware 63 / 85 / 93; and L1 75 / 91 / 98. Outer CV selected those candidates in 0 / 43 / 75, 24 / 53 / 78, and 62 / 83 / 87 months, respectively. The deterministic route and P&L remain valid, but these candidates behaved as one reachable state, weakening the two-regime interpretation. Scaled DD selected collapsed fits in 0 / 44 / 63 of 194 / 193 / 177 US / DE / JP months, so the same warning remains important.

Return prediction was tested, not postponed: the return-aware model was nearly identical to fixed JM and never improved Sharpe. This weakens that exact γ = 1 formulation; it does not prove that all predictive regime models fail.

Protocol, limitations, and evidence links

Reproducibility status: the fixed baseline is runnable, and the shared runner now reproduces and verifies the five simple challengers. The fresh run reproduced 24 metric rows, 45 traces, and 9 collapse summaries. The scale run separately verified 15 metric rows, 9 complete loss-to-trade traces, and 3 collapse summaries.

  • HMM input: equity log return. JM features: downside deviation DD-10 and Sortino-20/60.
  • Rolling model window: 3,000 observations. JM refits in January and July; HMM refits daily.
  • Monthly selection: preceding eight calendar years.
  • Frozen λ grid: 0, 5, 15, 35, 70, 150, 300, 600, 1200.
  • The five simple ideas and their order were frozen before their performance was opened. This remains exploratory development evidence, not a confirmatory study.
  • The evaluated proxy sample begins in 2007–2009, not the paper's exact 1990–2023 Bloomberg/GFD sample. Exact paper Sharpe is not the target.
  • The same through-2023 markets have been inspected repeatedly; no generalization or holdout claim is made. No reported model or P&L experiment used later rows, but a separate source audit inspected public candidate series through July 2026.
