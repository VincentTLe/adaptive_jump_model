Reuse policy

Use media because it teaches, not because space is empty

Generated illustration

Record purpose, date, tool, prompt summary, local hash, and the label “not data or evidence.”

External media

Verify title, author, source, license, and modification status at the canonical source. Search filters are not proof of permission.

Compatible sources require explicit reuse terms such as CC BY, CC BY-SA, CC0, public domain, or recorded written permission. Generic finance stock photography is outside the teaching contract.
