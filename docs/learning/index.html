What this course is for

Simple does not mean short

Each chapter begins with a small dollar example, introduces one idea at a time, defines every new term, and only then opens the mathematical layer. Worked examples, failure cases, interactive laboratories, code maps, paper citations, limitations, exercises, and advisor explanations make the material long enough to learn rather than merely summarize.

Course design

Visual-first, sourced, and useful offline

Each chapter opens in a shorter Visual path and can restore the Full chapter without deleting any authored explanation. MathJax enhances native equations and Chart.js enhances toy-data charts. Every calculation remains available through text, MathML, and visible tables when external scripts or all JavaScript are unavailable. Browser dependency versions and integrity hashes are recorded in browser-dependencies.json.

Generated and reused media are governed by the media credits and source-rights register. A concept illustration is never presented as market data, a paper figure, or project evidence.

Each chapter ends with recall, calculation, error diagnosis, cumulative review, and a short advisor explanation. The answers are part of the lesson because practice without feedback can reinforce the wrong model.
