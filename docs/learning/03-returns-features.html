Question 3

Did the model look into the future?

A realistic backtest lets each decision use only information that existed at that moment.

Remember this: a good formula becomes invalid evidence if even one future value enters a past decision.

A $100 example

A market investment rises from $100 to $102 in one day. Cash would have grown to $100.10.

Market return2.00%
Cash return0.10%
Excess return1.90%

A return is the percentage change in wealth. Excess return is market return minus the return available from cash.

Three model inputs, explained plainly

Downside deviation

How large have recent negative excess returns been?

Fast Sortino

Has recent excess reward been good relative to harmful moves?

Slow Sortino

What does the same balance look like over a longer memory?

The project uses exponentially weighted calculations: recent days matter more, while older days fade gradually. The downside memory has a 10-day half-life; the two Sortino memories use 20 and 60 days.

The timing rule

End of day tObserve data and produce a state
Day t+1Execution delay; no claimed benefit yet
Day t+2The state controls this return

This delay prevents the backtest from pretending that a signal calculated at the close was traded before that same close.

Past-only scaling

The three inputs have different units. Standardization puts them on comparable scales by subtracting a past average and dividing by past variability. The current run fits that scaler only on the trailing 3,000 observations.

Leakage: using the full dataset to standardize an old date would reveal future averages and future variability. That is forbidden, even if the model code never directly reads tomorrow's return.
Optional deeper look: the formulas

Simple return is P(t) / P(t-1) - 1. Log return is log(P(t) / P(t-1)). The strategy accounting uses simple total returns; the paper's HMM is fitted to log total returns. Sortino divides an exponentially weighted excess-return average by an exponentially weighted downside deviation. No outlier clipping is used in the frozen protocol.
