# Frozen Legacy Minute Pipeline

Status: frozen and unsupported. These files are retained only for research
provenance and are not part of the active daily research stack.

The archived scripts implement the earlier Kibot minute/tick experiments,
synthetic demonstration, and third-party backtest audit. That work used
hand-set penalty scales, did not implement the paper's daily walk-forward
protocol, and contains accounting and cache-reuse limitations documented in
the project audit.

Do not use these scripts for current results. Their generated data and reports
are intentionally excluded from Git; prior tracked outputs remain available in
Git history. The supporting legacy package code and tests are archived in
separate, reviewable cleanup commits.
